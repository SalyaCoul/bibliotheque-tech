🎯 En bref

WebSocket est un protocole de communication bidirectionnel et persistant entre un client et un serveur. Contrairement à HTTP (une requête → une réponse), WebSocket ouvre un canal permanent : le serveur peut envoyer des données au client à tout moment, sans que le client n'ait à demander.

Analogie

HTTP c'est comme sonner à la porte à chaque fois que vous voulez de l'information. WebSocket c'est comme laisser la porte ouverte et avoir une conversation en continu — les deux côtés peuvent parler quand ils veulent, sans avoir à frapper à chaque échange.

Utilisé par

Slack, Discord (chat temps réel), applications de trading (cours de bourse), Google Docs (édition collaborative), jeux en ligne multijoueurs, systèmes de notification push.

HTTP vs WebSocket

HTTP (Requête/Réponse) :
Client ──── GET /data ────→ Serveur
Client ←─── 200 OK ────── Serveur
Client ──── GET /data ────→ Serveur  (re-demande si nouvelles données)
Client ←─── 200 OK ────── Serveur

WebSocket (Canal persistant) :
Client ──── Handshake ────→ Serveur   (connexion établie une seule fois)
       ←──────────────────             Serveur envoie quand il veut
Client ──────────────────→             Client envoie quand il veut
       ←──────────────────             Données en temps réel
       ...connexion maintenue indéfiniment...

Cycle de vie d'une connexion WebSocket

// Côté client (navigateur)
const socket = new WebSocket('wss://api.example.com/ws');

// Connexion établie
socket.addEventListener('open', () => {
    console.log('Connecté au serveur');
    socket.send(JSON.stringify({ type: 'subscribe', channel: 'BTC/USD' }));
});

// Réception d'un message du serveur
socket.addEventListener('message', (event) => {
    const data = JSON.parse(event.data);
    console.log('Nouveau prix BTC :', data.price);
    updatePriceDisplay(data.price);
});

// Fermeture ou erreur
socket.addEventListener('close', () => console.log('Déconnecté'));
socket.addEventListener('error', (err) => console.error('Erreur :', err));
// Côté serveur (Node.js avec ws)
const WebSocket = require('ws');
const wss = new WebSocket.Server({ port: 8080 });

wss.on('connection', (ws) => {
    console.log('Nouveau client connecté');

    // Envoyer des données toutes les secondes (ex : cours en temps réel)
    const interval = setInterval(() => {
        ws.send(JSON.stringify({
            price: 42000 + Math.random() * 1000,
            timestamp: Date.now()
        }));
    }, 1000);

    ws.on('close', () => clearInterval(interval));
});

WebSocket✅ Oui (full-duplex)MoyenneChat, jeux, trading, collaboration
