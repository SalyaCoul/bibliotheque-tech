Amazon Simple Queue Service (SQS)

Statut
Terminé
TypeMessage Broker

🎯 En bref

Amazon SQS (Simple Queue Service) est un service de file d'attente de messages entièrement géré par AWS. Il permet à des composants d'une application de communiquer de façon asynchrone et découplée : l'émetteur dépose un message dans la file, le consommateur le traite quand il est disponible — sans que les deux aient besoin d'être disponibles en même temps.

Analogie

C'est comme une boîte aux lettres d'entreprise : le livreur (producteur) dépose les colis quand il veut, le bureau de tri (consommateur) les traite à son rythme. Si le bureau est débordé, les colis attendent dans la boîte sans que le livreur soit bloqué.

Utilisé par

Amazon lui-même, Netflix, Dropbox, et toute architecture AWS qui a besoin de découpler des microservices ou de lisser des pics de charge.

Fonctionnement d'une file SQS

┌──────────────┐    Envoie un message    ┌─────────────────────┐
│  Producteur  │ ───────────────────────→ │    File SQS         │
│ (Lambda, EC2,│                          │                     │
│  API, etc.)  │                          │  [msg1] [msg2] ...  │
└──────────────┘                          └──────────┬──────────┘
                                                     │ Reçoit et traite
                                                     ↓
                                          ┌─────────────────────┐
                                          │    Consommateur     │
                                          │ (Lambda, EC2, ECS)  │
                                          └─────────────────────┘

Exemple d'utilisation (Node.js + AWS SDK)

import { SQSClient, SendMessageCommand, ReceiveMessageCommand, DeleteMessageCommand } from "@aws-sdk/client-sqs";

const client = new SQSClient({ region: "eu-west-1" });
const QUEUE_URL = "https://sqs.eu-west-1.amazonaws.com/123456/ma-file";

// Producteur : envoyer un message
async function sendOrder(order) {
    await client.send(new SendMessageCommand({
        QueueUrl: QUEUE_URL,
        MessageBody: JSON.stringify(order),
        MessageAttributes: {
            OrderType: { DataType: "String", StringValue: "express" }
        }
    }));
    console.log("Commande envoyée dans la file");
}

// Consommateur : recevoir et traiter
async function processOrders() {
    const response = await client.send(new ReceiveMessageCommand({
        QueueUrl: QUEUE_URL,
        MaxNumberOfMessages: 10,
        WaitTimeSeconds: 20 // Long polling
    }));

    for (const message of response.Messages || []) {
        const order = JSON.parse(message.Body);
        await processOrder(order); // Votre logique métier

        // Supprimer le message après traitement
        await client.send(new DeleteMessageCommand({
            QueueUrl: QUEUE_URL,
            ReceiptHandle: message.ReceiptHandle
        }));
    }
}
