🎯 En bref

Xamarin est une plateforme Microsoft qui permet de développer des applications mobiles iOS, Android et Windows en C# avec une base de code partagée. Il donne accès aux APIs natives de chaque plateforme via des bindings C#, avec une UI soit partagée (XAML via .NET MAUI), soit native par plateforme.

Analogie

C'est comme avoir un chef français qui cuisine aussi bien des sushis que des tapas : il parle à la cuisine japonaise et espagnole dans leur propre langue (APIs natives), mais utilise ses propres techniques (C#) pour tout coordonner depuis une seule cuisine centrale.

Utilisé par

Entreprises dans l'écosystème Microsoft (.NET), Olo, Alaska Airlines — aujourd'hui en transition vers .NET MAUI (le successeur officiel de Xamarin depuis 2023).

Exemple de code partagé (C#)

// ViewModel partagé entre iOS et Android
public class UserViewModel : INotifyPropertyChanged
{
    private string _name;
    public string Name
    {
        get => _name;
        set
        {
            _name = value;
            OnPropertyChanged(nameof(Name));
        }
    }

    public ICommand LoadUserCommand { get; }

    public UserViewModel(IUserService userService)
    {
        LoadUserCommand = new Command(async () =>
        {
            var user = await userService.GetCurrentUserAsync();
            Name = user.Name;
        });
    }

    public event PropertyChangedEventHandler PropertyChanged;
    protected void OnPropertyChanged(string propertyName) =>
        PropertyChanged?.Invoke(this, new PropertyChangedEventArgs(propertyName));
}
