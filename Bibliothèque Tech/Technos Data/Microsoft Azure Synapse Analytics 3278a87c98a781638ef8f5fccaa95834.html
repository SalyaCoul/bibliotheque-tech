🎯 En bref : Microsoft Azure Synapse Analytics est une plateforme analytique unifiée qui combine Data Warehouse, Big Data et analyses en temps réel au sein d'un seul service Azure. Elle permet d'interroger des données structurées et non structurées via SQL ou Spark, avec une intégration native à l'écosystème Azure.

Analogie : Imagine un couteau suisse analytique. Avant, tu avais besoin d'un outil pour le Data Warehouse, un autre pour le Big Data, un autre pour le streaming. Synapse Analytics, c'est tout ça dans un seul espace de travail, connecté nativement à Azure Data Lake, Power BI et Azure ML.

Utilisé par : Data engineers, data analysts, équipes BI dans les entreprises fortement intégrées à Microsoft Azure qui veulent unifier leurs charges de travail analytiques.

Synapse Analytics est la solution de data warehouse cloud d'Azure. Il propose une intégration étroite avec d'autres services Azure et prend en charge des fonctionnalités telles que le traitement massivement parallèle (MPP) et l'analyse en temps réel.

Fonctionnalités clés

  • Synapse SQL : Moteur SQL distribué (MPP) pour interroger des données structurées à grande échelle.
  • Apache Spark intégré : Traitement Big Data directement dans Synapse, sans cluster externe.
  • Synapse Link : Connexion en temps réel avec Azure Cosmos DB pour des analyses hybrides OLTP/OLAP.
  • Intégration Azure : Pipeline natif avec Azure Data Factory, Power BI, Azure ML et Azure Data Lake Storage Gen2.
  • Serverless SQL Pool : Interroge des fichiers Parquet/CSV directement sur le Data Lake, sans charger les données.
