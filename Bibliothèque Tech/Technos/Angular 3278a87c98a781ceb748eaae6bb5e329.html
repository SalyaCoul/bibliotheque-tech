Angular

Statut
Terminé
TypeFramework

🎯 En bref

Angular est un framework frontend complet développé par Google, basé sur TypeScript. Contrairement à React (bibliothèque) ou Vue, Angular est un framework "batteries included" : routing, formulaires, HTTP, state management, tests — tout est intégré et structuré dès le départ.

Analogie

React c'est comme un couteau — précis, léger, vous choisissez vos outils autour. Angular c'est comme une cuisine professionnelle complète : tout est là, organisé, aux normes, prêt pour travailler en équipe dans les règles de l'art.

Utilisé par

Google (Gmail, Google Cloud Console), Microsoft, Deutsche Bank, Forbes, Santander — fortement adopté dans les grandes entreprises.

Exemple de composant Angular

// user-card.component.ts
import { Component, Input, OnInit } from '@angular/core';
import { UserService } from '../services/user.service';
import { User } from '../models/user.model';

@Component({
  selector: 'app-user-card',
  template: `
    <div *ngIf="user; else loading">
      <h2>{{ user.name | uppercase }}</h2>
      <p>Inscrit le : {{ user.createdAt | date:'dd/MM/yyyy' }}</p>
      <button (click)="delete()">Supprimer</button>
    </div>
    <ng-template #loading>Chargement...</ng-template>
  `
})
export class UserCardComponent implements OnInit {
  @Input() userId!: string;
  user?: User;

  constructor(private userService: UserService) {} // Injection de dépendance

  ngOnInit(): void {
    this.userService.getById(this.userId).subscribe(u => this.user = u);
  }

  delete(): void {
    this.userService.delete(this.userId).subscribe();
  }
}
