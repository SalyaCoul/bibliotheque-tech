🎯 En bref

Kotlin est un langage moderne développé par JetBrains qui tourne sur la JVM (Java Virtual Machine), entièrement interopérable avec Java. Il est le langage officiel de Google pour Android depuis 2017 et se distingue par sa concision, sa null-safety intégrée et ses coroutines pour l'async.

Analogie

Java c'est comme conduire une voiture des années 90 : fiable, mais beaucoup de gestes manuels. Kotlin c'est la même voiture avec boîte automatique, parking assisté et régulateur de vitesse — même moteur (JVM), mais l'expérience est bien meilleure.

Utilisé par

Google (Android, backend), Pinterest, Trello, Uber, Slack — et la quasi-totalité des nouveaux projets Android depuis 2019.

Kotlin vs Java — Comparaison syntaxique

// Java
public class User {
    private String name;
    private int age;
    public User(String name, int age) { this.name = name; this.age = age; }
    public String getName() { return name; }
    public int getAge() { return age; }
    // + hashCode, equals, toString...
}

// Kotlin — Data class équivalente (1 ligne !)
data class User(val name: String, val age: Int)
// Génère automatiquement : constructeur, getters, equals, hashCode, toString, copy()

Exemple pratique — Kotlin idiomatique

// Null safety
val name: String = "Alice"    // Non-nullable, ne peut pas être null
val nickname: String? = null  // Nullable explicite

val length = nickname?.length ?: 0  // Elvis operator : 0 si null

// Extension function
fun String.capitalize(): String = this.replaceFirstChar { it.uppercase() }
val result = "hello world".capitalize() // "Hello world"

// Data class + copy
data class User(val name: String, val age: Int, val active: Boolean = true)
val alice = User("Alice", 30)
val olderAlice = alice.copy(age = 31) // Copie immuable avec modification

// Sealed class pour les états
sealed class ApiResult<T> {
    data class Success<T>(val data: T) : ApiResult<T>()
    data class Error<T>(val message: String) : ApiResult<T>()
    class Loading<T> : ApiResult<T>()
}

// When expression (exhaustif)
fun handleResult(result: ApiResult<User>) = when (result) {
    is ApiResult.Success -> println("Utilisateur : ${result.data.name}")
    is ApiResult.Error   -> println("Erreur : ${result.message}")
    is ApiResult.Loading -> println("Chargement...")
}

Coroutines — Async simplifié

import kotlinx.coroutines.*

// suspend : fonction qui peut être suspendue sans bloquer le thread
suspend fun fetchUser(id: Int): User {
    delay(1000) // Simule un appel réseau (non-bloquant)
    return User("Alice", 30)
}

// Lancement dans un scope coroutine
fun main() = runBlocking {
    // Lancement parallèle
    val user1 = async { fetchUser(1) }
    val user2 = async { fetchUser(2) }

    println("${user1.await().name} et ${user2.await().name}")
}
