🎯 En bref : Expo est une surcouche au-dessus de React Native qui simplifie radicalement le développement d'applications mobiles. Là où React Native "nu" demande de configurer Xcode, Android Studio et des outils complexes, Expo permet de démarrer en 5 minutes.

Analogie : React Native, c'est construire une maison depuis les fondations. Expo, c'est acheter un kit maison préassemblé avec toutes les pièces déjà taillées, les outils inclus, et des instructions claires. Vous pouvez customiser chaque détail si besoin, mais la majorité des cas courants sont déjà gérés.

Utilisé par : Startups qui veulent aller vite, équipes web qui s'aventurent sur mobile, applications de taille petite à moyenne. Des milliers d'apps dans l'App Store et le Play Store ont été construites avec Expo.

Expo

Expo est un framework et une plateforme basés sur React Native pour créer des applications iOS, Android et web avec JavaScript/TypeScript.

Ce qu'Expo apporte

1. Démarrage instantané — Pas besoin d'installer Xcode ou Android Studio pour commencer à développer :

npm install -g expo-cli
npx create-expo-app mon-app
cd mon-app && npx expo start

Scannez le QR code avec votre téléphone → l'app s'ouvre directement !

2. Expo Go — Une app sur votre téléphone qui permet de tester votre code en temps réel, sans build ni déploiement. Parfait pour le développement rapide.

3. SDK Expo — Une collection de modules prêts à l'emploi pour les fonctionnalités natives courantes :

  • expo-camera : accès à la caméra
  • expo-location : géolocalisation
  • expo-notifications : notifications push
  • expo-image-picker : sélection de photos
  • expo-auth-session : authentification OAuth

4. EAS (Expo Application Services) — Service cloud pour builder et déployer vos apps sans avoir de Mac pour publier sur l'App Store.

Expo Router (v2+)

Expo intègre désormais un système de routing basé sur les fichiers (comme Next.js), permettant de créer des apps mobile et web partageant le même code de navigation.

app/
  index.tsx        → écran d'accueil
  profile.tsx      → /profile
  settings/
    index.tsx      → /settings
💡 Expo a considérablement mûri — la majorité des apps React Native professionnelles peuvent être construites avec Expo aujourd'hui. La question n'est plus "Expo ou React Native" mais "Expo Managed Workflow ou Expo Bare Workflow" selon vos besoins.
