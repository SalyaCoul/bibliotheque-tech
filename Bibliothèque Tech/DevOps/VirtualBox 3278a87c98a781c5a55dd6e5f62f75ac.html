🎯 En bref

VirtualBox est un logiciel de virtualisation de type 2 open source développé par Oracle. Il permet de créer et gérer des machines virtuelles (VM) sur votre ordinateur — des ordinateurs complets émulés logiciellement, chacun avec son propre OS, ses ressources et son réseau, qui s'exécutent dans une fenêtre sur votre machine principale.

Analogie

C'est comme avoir plusieurs ordinateurs dans votre ordinateur. Vous êtes sur Windows, et dans une fenêtre vous avez Ubuntu qui tourne, dans une autre macOS — chacun complètement isolé, comme s'il était sur une vraie machine physique. Si vous cassez l'un, les autres continuent de tourner.

Utilisé par

Développeurs pour tester sous différents OS, équipes DevOps pour simuler des environnements serveur en local, formateurs pour créer des environnements d'apprentissage reproductibles, testeurs QA.
